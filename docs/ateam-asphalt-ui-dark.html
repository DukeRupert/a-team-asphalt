04 — Brand Usage Notes

✓ Do

  • Use Asphalt Black as the primary background — always dark-first
  • Use Team Red for primary CTAs, Chrome Silver for secondary accents — pair them
  • Use Oswald 700 for display headlines, 600/500 for subheads
  • Use the star motif from the logo as a corner or section accent
  • Use the red → silver → charcoal stripe pattern on divider bars
  • Keep the A-Team nod playful but professional — trust-first

✕ Don't

  • Don't use red as a background fill — accent only
  • Don't use light/white backgrounds — this is a dark brand
  • Don't add a third font family — Oswald + Source Sans 3 only
  • Don't lean so hard into the TV reference it feels campy
  • Don't use rounded buttons — the clipped corners are signature
  • Don't use thin font weights for headlines
  • Don't use the standard logo on dark without inversion — create a white variant with red A
