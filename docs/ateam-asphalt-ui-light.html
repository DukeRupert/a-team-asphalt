05 — Light Mode Usage Notes

✓ Do

  • Keep hero sections dark — the brand anchors in dark steel
  • Use the chrome gradient panel for stats and feature callouts
  • Steel 50 background with white cards creates clean depth
  • Team Red remains the only warm color — use it as a signal
  • The red corner swoop works well on light service cards
  • Light mode suits content-heavy pages: service listings, blog, about

✕ Don't

  • Don't go pure white — always use the steel-tinted base
  • Don't soften the red — it should still feel bold, not pastel
  • Don't use the light mode for a full site — hero/CTA stays dark
  • Don't add warm tones (tan, beige, yellow) — keep it industrial
  • Don't use rounded corners or soft shadows — precision only
  • Don't drop the clipped button corners in light mode
